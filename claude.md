# Agent Context — Big Tech Club

Single-page speculative essay at `index.html`, hosted on GitHub Pages at bigtechclub.com (`CNAME` in repo root). The essay argues for a minimal developer identity platform: a calm landing page (photo, bio, links) with an optional structured layer (self-reported skills + level tags) that makes engineers discoverable by location, technology, and seniority. It is written for developers but doubles as an implicit pitch to **Boot.dev** — a dev education platform whose community already has inbound employer hiring interest. Boot.dev is never named; Part IV's closing paragraphs are the operative pitch.

Design is inspired by maggieappleton.com: Lora serif body, Inter sans labels, warm off-white background (`#faf9f7`), `75ch` prose column, generous line height. Self-contained HTML/CSS, no build step.
